# Banking Management System (Core Java)

This is a basic console-based banking management system developed in Core Java.  
It demonstrates Object-Oriented Programming concepts and allows users to:

- Create a bank account
- Deposit money
- Withdraw money
- Check account balance

## How to Run

1. Open terminal/command prompt in `src/` folder.
2. Compile all Java files:
   ```bash
   javac bank/*.java
